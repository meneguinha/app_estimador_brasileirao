# ⚽ Preditor de Público e Renda — Brasileirão

Aplicação web interativa construída com **Streamlit** para prever **público pagante** e **renda de jogos** do Brasileirão Série A com base em dados históricos.

---

## 📌 Visão Geral

Este projeto utiliza **Machine Learning (Gradient Boosting)** para estimar:

- 👥 Público pagante esperado  
- 💰 Renda estimada  

As previsões consideram fatores como:

- Time mandante e visitante  
- Dia da semana  
- Horário da partida  
- Ano  
- Médias históricas do mandante
